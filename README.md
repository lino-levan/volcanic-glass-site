# Volcanic Glass

Volcanic Glass is a project by a bunch of open source developrs which adds sharing functionality to [Obsidian MD](https://obsidian.md). The public-facing website uses create-next-app.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `pages/index.js`. The page auto-updates as you edit the file.
